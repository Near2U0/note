# switch

```c
switch (控制表达式){
  case 常量:
    语句
  case 常量:
    语句
  //...
  default:
    语句
      //...
}

/*
1.控制表达式只能是整数型的结果
2.常量可以是常数，也可以是常数计算的表达式
3.建议case中加上break，不然程序会顺序执行到下一个case，直到遇到break，或者switch结束为止
*/
```
