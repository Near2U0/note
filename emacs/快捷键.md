# emacs 快捷键

* Ctrl--C
* Alt--M

## 翻页

```shell
#向前翻页
C+V  

#向后翻页
M+V

#来回切换半页
C+L

```

## 行间移动和行内移动

```shell
#P--previous（上一行）
#N--next（下一行） 
#B--next(下一行) 
#F--backward(回退) 

#上一行
C-P

#下一行
C-N

#前一个字符
C-f

#后一个字符
C-b

#移动多个词
M-f 向前移动多个词
M-b 向后移动多个词

#移动到开头，结尾
C-a
C-e

#移动到句首，句尾
M-a
M-e

#移动到文字的最开头，移动到文字的结尾处
M-< (小于号：用shift)  
M->




```
