# 静态链表

链表是内存上非连续的存储结构，由一系列节点组成，节点可以在运行时动态生成，节点包括：数据域和指针域

```c
typedef struct Teacher
{
    int data;
    struct Teacher *next;
}Teacher;

Teacher * Create3Teacher()
{
    Teacher *tmp = NULL;
    Teacher t1, t2, t3;
    t1.data = 1;
    t2.data = 2;
    t3.data = 3;
    
    //构建链表：固定节点个数，静态链表
    t1.next = &t2;
    t2.next = &t3;
    t3.next = NULL;
    
    //遍历链表
    tmp = &t1;
    while(tmp->next != NULL)
    {
        printf("data=%d\n", tmp->data);
        tmp = tmp->next;//向后移动
    }
    
    return &t1;
}

void main(void)
{

    //error，节点是在方法中创建的，方法调用之后，会释放，那么创建的节点也就会释放
    Teacher * head = Create3Teacher();
    
}
```
