# 标识符的作用域与可见性

- 作用域是一个标识符在程序正文中有效的区域。
- 作用域分类

1. 函数原型作用域
2. 局部作用域(块作用域)
3. 类作用域
4. 文件作用域
5. 命名空间作用域（详见第10章）

## 函数原形作用域

- 函数原型中的参数，其作用域始于"("，结束于")"。
- 函数原形作用域举例

double area(double radius);

## 局部作用域

- 函数的形参、在块中声明的标识符；
- 其作用域自声明处起，限于块中。
