# 数组

```c
int main()
{
  int array[10];//定义了一个一维数组，10个元素（[0]-[9]），他们的类型相同
}
//数组在内存中就是一段连续的空间，每个元素所占用的内存大小是一样的

//初始化,定义数组的同时为数组的成员初始化值
int arr[10] = {0,1,2,3,...,9};
int arr[10] = {2,3,5};//未初始化的元素，自动赋值为0
int arr[10] = {0};//将数组的所有元素都置为0
```

# 二维数组

```c
int arr[2][3];//定义了一个二维数组，有两个arr[3]

arr[0][0] = 0;//第0行第0列赋值

//初始化,定义二维数组的同时，初始化成员
int arr[2][3] = {{1,2,3},{0},{0}}

int arr[2][3] = {0}//将二维数组中所有的元素初始化为0
```
