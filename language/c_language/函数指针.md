# 函数指针

```c
//Function Pointer 指向函数的指针变量
/*
数据类型 (*指针变量名)(形参列表);   //数据类型 和 (形参列表) 唯一的确定一个函数， 指针变量 就是指向的这种类型

(*指针变量名) 括号不能丢
*/


int Fun(int a, int b);
//定义函数指针
int (*funp)(int, int);
funp = Fun;//让funp指向函数Fun();
/**
编译器将不带()的函数名解释为该函数的入口地址
函数指针存储的是函数在内存中的入口地址
*/


(*funp)(3, 4);//调用
```
