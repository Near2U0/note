# 段

段的定义：section， segment，如果没有使用，默认所有的程序是一个段，这就是默认段

段的起始和结束：下一个段的开始就是上一个段的结束

段属性

* align：为了构造16字节对齐，不够的需要填充0

```shell
section.段名称.start
#段header相对于整个程序的开头的汇编地址是section.header.start
```

* vstart=0 相对于该段的起始偏移，否则就是相对于程序开始的位置的偏移量
