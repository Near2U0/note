操作系统启动必须

系统启动就需要用到的程序，这些目录不能挂载额外的分区（因为在这些目录下挂载分区的话，原来目录下的文件就会被暂时隐藏），所以必须要在 / 分区上

```
/etc,  
/bin, 
/sbin,  
/lib
/etc

```





操作系统提供的核心服务需要的依赖

```
/usr/
	bin
	sbin
	lib
	etc
```



第三方软件所依赖的目录，这个是一个独立王国，可以将该目录放到一个分区中

```
/usr/local
	bin
	sbin
	lib
	etc
	man
```



辅助目录

```
/proc	
/sys
#这两个不能单独分区，默认为空

/dev 设备：不能单独分区

/var 用来存放 日志文件，锁文件，邮件等

/boot 内核，建议单独分区
```
