## 输入，输出重定向



```
>  : 表示正确覆盖输出
>> : 正确追加输出

2> : 错误覆盖输出
2>> : 错误追加输出
&> 重定向标准输出或错误输出到同一个文件

< : 输入重定向
cat < /etc/profile

<< ： Here Document

id student &> /dev/null   输出到null

```
