# 快捷键



| 移动光标的方法 |                                                              |
| -------------- | ------------------------------------------------------------ |
| h              | 左移                                                         |
| j              | 下                                                           |
| k              | 上                                                           |
| i              | 右移                                                         |
| 30j；其他同理  | 向下移动30行                                                 |
| ctrl+f         | 向下移动一页                                                 |
| ctrl+b         | 向前翻一页                                                   |
| n<space>       | 那个 n 表示“数字”，20<space> 则光标会向后面移动 20个字符距离 |
| 0              | 数字0：移动到这一行的最前面字符                              |
| $              | 移动到这一行的最后一个字符                                   |
| H              | 移动到屏幕最上方第一行的第一个字符                           |
| M              | 移动到屏幕中间的一行的第一个字符                             |
| L              | 移动到屏幕最下方一行的第一个字符                             |
| G              | 移动到这个文件的最后一列                                     |
|                |                                                              |
|                |                                                              |
|                |                                                              |
