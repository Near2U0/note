# create new branch

```shell
# show all branch
git branch -a 


# create new branch
#refrence:https://www.git-tower.com/learn/git/faq/create-branch/
git branch <new-branch>

# checkout new branch
git checkout dev-cys

# show current branch and push current branch
git status
git add .
git commit -m "开发branch-dev-cys"
git push
git push --set-upstream origin dev-cys

```
